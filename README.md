# Vue-spa-prototype

A Single Page Application prototype built with Vue.js 3, demonstrating basic routing, layout structure, and navigation patterns.

This repository is intended as a practice/starter template for building Vue-based SPAs with simple page transitions and component structure.
use Vue Vue Router, typescript

```
https://vue-spa-prototype-a99c6.firebaseapp.com/
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
